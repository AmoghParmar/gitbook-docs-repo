# Naming Conventions

## File Naming Rules

Use lowercase letters and hyphens in file names.

## Examples

Correct:
- quick-start.md
- publish-guide.md

Incorrect:
- QuickStart.md
- Publish Guide.md

## Folder Naming

Folders should also follow consistent naming conventions.

## Why Naming Matters

Consistent naming improves:
- Readability
- Searchability
- Organization
