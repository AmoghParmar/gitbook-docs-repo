# Troubleshooting

## Sync Issues

Sometimes GitBook may not immediately sync with GitHub.

## Solutions

- Refresh GitBook
- Push latest commits
- Check repository permissions

## Navigation Problems

If navigation does not appear:
- Verify SUMMARY.md
- Check file paths
- Ensure markdown files exist

## Publishing Errors

If publishing fails:
- Confirm public access
- Reconnect GitHub repository
- Retry publishing

## Conclusion

Most GitBook issues can be solved by checking synchronization and file structure.
